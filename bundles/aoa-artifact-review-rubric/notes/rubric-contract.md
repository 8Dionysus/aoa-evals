# Rubric Contract

This starter bundle stays on artifact quality only.

Canonical v1 rubric axes:
- task fit to the visible request
- internal coherence and obvious defect load
- bounded completeness on the visible task surface
- reviewability and readability of the produced artifact

Public summary discipline:
- keep axis-level notes visible
- do not collapse the result into a scalar score
- keep the bundle-level verdict weaker than the strongest-looking single axis
- do not upgrade artifact polish into workflow proof

If the main question is whether artifact polish is outrunning workflow discipline,
use `aoa-output-vs-process-gap`.
