# Provenance Contract

Use this bundle when a witness-facing input is being distilled into a compost artifact.

The artifact should keep these surfaces explicit:
- source refs back to the witness-facing input
- review state
- current limits
- promotion boundary
- contradiction,
  stale posture,
  or demotion possibility

The artifact may become cleaner than the raw input.
It should not become cleaner by severing the path that made it trustworthy in the first place.
