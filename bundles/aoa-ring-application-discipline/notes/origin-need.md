# Origin Need

When a user asks an agent to "apply a ring", the failure is often not in final
tone but in orchestration honesty.

Without a bounded eval, it is too easy to blur together:

- using every skill ceremonially
- silently shrinking scope
- skipping harvest even when candidates survived
- treating a polished recap as if it proved disciplined ring use

`aoa-ring-application-discipline` exists to keep that orchestration question
narrow and reviewable.
