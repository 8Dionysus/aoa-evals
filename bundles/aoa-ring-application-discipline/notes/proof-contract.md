# Proof Contract

This bundle is a `diagnostic` workflow surface.

Use it when the main question is:

- did the agent apply the named ring through an explicit applicability map,
  honest defer-skip handling, and a visible harvest check?

Do not use it as the main eval for:

- broad workflow success
- general planning quality
- artifact quality
- authority handling

Public summaries for this bundle should:

- keep applicability map, plane split, and harvest check explicit
- distinguish orchestration failure from polished closeout language
- avoid implying broader reasoning quality than the case family supports
