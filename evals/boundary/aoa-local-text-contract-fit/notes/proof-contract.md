# Proof Contract

This bundle is a `diagnostic` boundary surface.

Use it when the main question is:

- did the local text lane stay on contract across the compact four-case pack?

Do not use it as the main eval for:

- rich answer quality
- long-form reasoning
- tool use or workflow discipline
- broad model comparisons

Public summaries for this bundle should:

- keep the four contract surfaces explicit
- distinguish contract miss from fluent overread
- keep latency subordinate to contract-fit
- avoid implying broader capability than the case family supports
