# Eval Integrity Check

Use `aoa-eval-integrity-check` as a sidecar when this bundle's report wording
starts to imply stronger claims than the supplied recurrence dossier can prove.

The especially risky overclaims are:

- treating registry tolerance as complete manifest correctness;
- treating graph closure as routing or KAG graph authority;
- treating beacons as owner decisions;
- treating stats counts as verdicts;
- treating Agon diagnostics as live trial, rank, scar, or spawn machinery;
- treating downstream projections as source truth.
