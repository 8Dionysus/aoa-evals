# Origin Need

The repository now has multiple one-run starter bundles,
but it still lacks a first public regression surface.

That makes it possible to ask:
- what happened on this run?

It does **not** yet make it possible to ask the next bounded question:
- did the candidate materially regress against a frozen baseline on the same task family?

`aoa-regression-same-task` exists to open Layer 2 carefully.
It keeps the comparison narrow:
- same bounded cases
- frozen baseline
- modest comparative verdicts

This bundle should expose same-task regression
without pretending to prove general capability growth or decline.
