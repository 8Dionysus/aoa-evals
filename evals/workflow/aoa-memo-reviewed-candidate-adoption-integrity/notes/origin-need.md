# Origin Need

`aoa-memo-reviewed-candidate-adoption-integrity` exists because the current
public memo proof cluster already covers recall, contradiction, and one
confirmed decision writeback act, but it still does not prove the stricter
reviewed-candidate path that the writeback-act pilot explicitly left open.

The missing question is no longer "can one confirmed decision land as memo?"
The missing question is "can reviewed distillation claim, pattern, and bridge
candidates be adopted into memo, published through the live receipt seam, and
recalled downstream without pretending promotion is already solved?"

This bundle tests that narrower adoption question before any broader promotion,
ledger, retention, or graph-lift story is promoted downstream.
