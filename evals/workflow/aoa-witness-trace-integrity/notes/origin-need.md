# Origin Need

This starter bundle exists because a witness route can fail quietly even when the final summary looks clean.

The public pilot pair needs a bounded proof surface that checks whether:
- meaningful steps stayed visible
- tool use was not hidden
- external effects were marked explicitly
- failure paths still left a partial witness
- redaction protected sensitive content without erasing reviewability

Without that surface,
the witness contract risks becoming a polished summary format instead of a reviewable trace artifact.
