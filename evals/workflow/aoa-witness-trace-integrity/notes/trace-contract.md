# Trace Contract

Use this bundle when the public witness artifact is the thing under review.

The trace should keep these surfaces explicit:
- meaningful steps
- tool visibility
- `state_delta` notes for external effects
- partial failure paths
- redaction posture
- markdown-summary alignment

The trace does not need to reveal hidden chain-of-thought or private payloads.
It does need to remain reviewable enough that downstream memo writeback and compost distillation do not float free from the actual route.
