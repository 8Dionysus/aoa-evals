# Antifragility Legacy

Legacy preserves old placement and path vocabulary behind the active
`mechanics/antifragility/` route.

Start from `../README.md` and `../PARTS.md`. Open this directory only when a
former root path, schema location, or shared fixture family path must be mapped
to the current active part.

## Required Route

Open `../PROVENANCE.md` before using this directory. Use `INDEX.md` for
old-path lookup, `DISTILLATION_LOG.md` for raw-to-active accounting, and
`raw/README.md` for raw lineage. Legacy is not active topology and not a
new-work entrypoint.
