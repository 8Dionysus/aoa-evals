# Comparison Spine Legacy

Legacy comparison-spine files preserve old root fixture placement history.

They are not the active comparison proof route. Start from
`mechanics/comparison-spine/README.md` and enter legacy through
`mechanics/comparison-spine/PROVENANCE.md` only when old path lineage matters.

## Required Route

Open `../PROVENANCE.md` before using this directory. Use `INDEX.md` for
old-path lookup, `DISTILLATION_LOG.md` for raw-to-active accounting, and
`raw/README.md` for raw lineage. Legacy is not active topology and not a
new-work entrypoint.
