# Experience Legacy

Legacy preserves old Experience root placement as provenance behind the active
mechanic.

Use this directory only to answer where an old root doc, schema, example,
fixture, or test went. New work belongs in the active part route.

## Required Route

Open `../PROVENANCE.md` before using this directory. Use `INDEX.md` for
old-path lookup, `DISTILLATION_LOG.md` for raw-to-active accounting, and
`raw/README.md` for raw lineage. Legacy is not active topology and not a
new-work entrypoint.
