# Experience Certification Eval Bundles

## Role

This source note names the bundle family used by the Experience certification
gate.

The certification judge evaluates gate completeness, regression results,
rollback evidence, compatibility claims, and authority validity.

## Reads

Use this surface when the part README points to certification-adjacent bundle
support and an agent needs to know what the bundle family is allowed to prove.

## Boundary

The bundle family supports readiness-for-operator-review proof. It does not
certify an assistant, approve release, promote rollout, or declare runtime
health.

## Validation

Follow `mechanics/experience/parts/certification-gate/README.md`.

Bundle-local reports should preserve the same bounded claim wording that the
judge reads.
