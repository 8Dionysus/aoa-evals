# Publication Receipts Parts

Part-local homes for publication receipt artifacts.

- `receipt-payload/`: bounded `eval_result_receipt` payload contract.
- `stats-envelope-mirror/`: local mirror of the canonical `aoa-stats` event envelope.
- `live-publisher/`: append tool for already reviewed and schema-valid receipt envelopes.
- `intake-dry-review/`: non-publishing receipt-intake review artifact.

Use the package root `README.md` for the operation route and `PARTS.md` for the
part boundary map.
