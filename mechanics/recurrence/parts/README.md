# Recurrence / Parts Route

`mechanics/recurrence/parts/` is the lower index for active eval-side
recurrence proof parts. Use it after the parent Recurrence route has selected a
return, memory-recall, control-plane, recursor, stats-regrounding, or
portable-proof operation and the next agent needs the exact part, payload home,
owner route, tool lane, and validation lane.
