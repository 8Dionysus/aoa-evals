# Recurrence Control Plane Evals

`aoa-recurrence-control-plane-integrity` is the first bounded proof surface for the recurrence program itself.
It checks recurrence artifacts without taking ownership of recurrence doctrine.

## Role

This note routes the recurrence control-plane bundle from source proof meaning
to the part-local dossier, runner, scorer, fixture, tests, and recurrence
manifest.

## Reads

Use it when recurrence evidence needs to be judged as a bounded control-plane
integrity packet, not as global recurrence completeness.

The bundle may inspect manifest scan, graph closure, hook run, beacon, review
decision, downstream projection, and Agon diagnostic evidence.

## Boundary

`aoa-evals` owns bounded proof wording, dossier schema expectations,
runner/scorer behavior, seeded fixture checks, and report interpretation.

Agents-of-Abyss, runtime, routing, agents, playbooks, downstream owners, and
Agon owner surfaces keep doctrine, runtime truth, live routing, handoff posture,
projection truth, and Agon source truth.
