# Live observation producer notes for evals

## Role

This note explains how live observation producer inputs may travel into
recurrence control-plane evals.

Producer inputs include runtime evidence-selection sidecars and runtime
promotion guides. Runtime posture can become selected evidence, but not portable
proof canon without owner review.

## Reads

Use this surface when a recurrence dossier cites live runtime observations,
evidence-selection sidecars, or promotion-guide output.

## Boundary

`aoa-evals` may read selected observation evidence as part of a bounded dossier.
Runtime owners keep live observation production, runtime status, logs, activation,
and self-healing truth.

## Validation

Route through
`mechanics/recurrence/parts/control-plane-integrity/README.md` and the runtime
candidate evidence guides before treating observation evidence as portable.
