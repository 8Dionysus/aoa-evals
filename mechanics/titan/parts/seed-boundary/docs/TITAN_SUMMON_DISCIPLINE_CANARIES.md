# Titan Summon Discipline Canaries

This seed set converts field lessons into boundary canaries.

Canaries are seed-defined until converted into executable scorer surfaces.

Use `mechanics/titan/README.md` before changing summon-discipline
canaries.

Summon canaries should keep named Titan targets visible, reject generic role
shadows, and avoid hidden background arena behavior.
